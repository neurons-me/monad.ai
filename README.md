# monad.ai

Monad.ai is the daemon host for the `me://` runtime.

It currently contains three package surfaces:

- `npm/` — Node daemon package and HTTP runtime
- `crate/` — Rust daemon/runtime work
- `pip/` — Python package surface

Recommended entry points:

- `npm/README.md`
- `npm/docs/nrp-routing-spec.md`
- `npm/docs/nrp-remote-exchange-spec.md`

The root `index.html` is the HTML landing page for this repository, mirroring the pattern used in `.me`.
